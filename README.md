# SFWRENG 2AA4 – Bonus Assignment 2

This repository contains the solution for **Bonus Assignment 2** in **SFWRENG 2AA4 (Software Design I)** at McMaster University.  
The assignment focuses on applying software design principles, abstraction, and clean structuring of code as taught in the course.

---

## Overview

The goal of this assignment is to:
- Practice good software design
- Apply abstraction and separation of concerns
- Write readable, maintainable, and well-structured code
- Demonstrate understanding of concepts discussed in lectures and tutorials

This project is intended for educational purposes only and follows the constraints and expectations set by the course.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/erika-goh/SFWRENG-2AA4-Bonus-Assignment-2.git
   ```

2. Move into the project folder:
   ```bash
   cd SFWRENG-2AA4-Bonus-Assignment-2
   
   ```

4. Run the program:
   ```bash
   python code_generator.py
   ```

> If `python` doesn’t work on your system, try:
```bash
python3 code_generator.py
```
